# From Scratch Codes

This Repository contains all my from scratch implemented algorithms, functions, and research papers.

The Implementations is categorized in the following folders as:
- **Research Papers** : Contains the implementation of various research papers from scratch.

- **ML_Algorithms** : Contains the implementation of various machine learning algorithms from scratch.

- **DL_Algorithms** : Contains the implementation of various deep learning algorithms from scratch.

- **CS_Algorithms** : Contains the implementation of various Data Structures and Algorithms from scratch.

- **Math_Fundamentals** : Contains the implementation of various mathematical concepts and functions from scratch.
